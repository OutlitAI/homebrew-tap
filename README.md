# Outlit Homebrew Tap

Homebrew formulae for [Outlit](https://outlit.ai) tools.

## Install

```sh
brew install outlitai/tap/outlit
```

Or tap first, then install:

```sh
brew tap outlitai/tap
brew install outlit
```

## Upgrade

```sh
brew upgrade outlit
```

## Brewfile

```ruby
tap "outlitai/tap"
brew "outlit"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
